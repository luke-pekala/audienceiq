Privacy

Last updated May 2026

What we collect

In the current heuristic version, AudienceIQ processes all text entirely in your browser. No audience descriptions or analysis results are sent to any server. Nothing is stored remotely.

Saved personas are stored in your browser's localStorage and never leave your device.

When the Claude API is enabled

The next version will optionally use the Anthropic Claude API to generate deeper analysis. When that feature is active, your audience description will be sent to Anthropic's API for processing. Anthropic's privacy policy will apply to that data. You will be notified clearly when API processing is enabled.

Analytics

This app does not use third-party analytics, tracking pixels, or advertising. There are no cookies beyond what the browser itself uses.
